# 紫微斗数技能

> "紫微斗数，星命之宗。" —— 传统命理学

紫微斗数是中国传统命理学中最精密的星命术数，以出生年月日时排布十四主星于十二宫位，推算命运、性格、事业、婚姻、财运等。本技能提供完整的紫微斗数排盘和命盘分析功能。

## ✨ 核心功能

- ✨ 十四主星排布（紫微、天机、太阳、武曲、天同、廉贞、天府、太阴、贪狼、巨门、天相、天梁、七杀、破军）
- ✨ 十二宫位定位（命宫、兄弟、夫妻、子女、财帛、疾厄、迁移、交友、官禄、田宅、福德、父母）
- ✨ 四化飞星推算（化禄、化权、化科、化忌）
- ✨ 五行局计算（水二局、木三局、金四局、土五局）
- ✨ 大限流年推算
- ✨ 命盘分析（性格、事业、婚姻、财运）
- ✨ 辅星排布（左辅、右弼、文昌、文曲、天魁、天钺等）
- ✨ 纯 Python 实现，无外部依赖

## 🔧 技术特点

- 🔧 精确农历转换（1900-2100 年）
- 🔧 精确节气计算
- 🔧 安星算法完整（14 主星 + 辅星 + 四化）
- 🔧 纯 Python 实现，无外部依赖
- 🔧 支持 JSON 格式输出
- 🔧 命令行界面友好

## 📖 快速开始

### 安装

```bash
# 克隆仓库
git clone https://github.com/dglijin-oss/ziwei-skill.git
cd ziwei-skill

# 确保 Python 3.8+ 已安装
python3 --version

# 赋予执行权限
chmod +x index.js
```

### 基本使用

```bash
# 基础排盘
node index.js --date "1990-05-15" --hour 8

# 指定性别
node index.js --date "1990-05-15" --hour 8 --gender 女

# 输出 JSON 格式
node index.js --date "1990-05-15" --hour 8 --json
```

### 命令行参数

| 参数 | 简写 | 说明 | 默认值 |
|------|------|------|--------|
| --date | -d | 出生日期 (YYYY-MM-DD) | 必填 |
| --hour | -H | 出生时辰 (0-23) | 必填 |
| --gender | -g | 性别 (男/女) | 男 |
| --json | -j | 输出 JSON 格式 | 否 |

## 🐍 Python API 调用

```javascript
const ziwei = require('./index.js');

// 排盘
const result = ziwei.ziwei_pan({
    date: "1990-05-15",
    hour: 8,
    gender: "男"
});

// 格式化输出
console.log(ziwei.format_output(result));

// 或直接获取字典数据
```
